  <behavior>
    - RecordingSession starts in .idle state
    - startRecording() transitions to .recording, creates AudioRecorder, starts elapsed timer
    - pause() transitions to .recording -> .paused, records pause timestamp
    - resume() transitions .paused -> .recording, records resume timestamp
    - stop() transitions to .stopped, stops recorder, moves temp file to final destination, returns RecordingResult
    - elapsedSeconds returns the total recording time excluding paused intervals
    - pauseIntervals returns [(start: TimeInterval, duration: TimeInterval)] for P-D1 inline markers
    - State machine rejects invalid transitions (e.g., pause when .idle)
  </behavior>
  <action>
    Create RecordingSession as a `public actor RecordingSession: Sendable` in Sources/UnibrainProviders/Capture/RecordingSession.swift.

    Define `public enum RecordingSessionState: Sendable, Equatable` with cases: `idle`, `recording`, `paused`, `stopped`.

    Define `public struct RecordingResult: Sendable` with: `audioURL: URL`, `durationSeconds: Int`, `pauseIntervals: [(start: TimeInterval, duration: TimeInterval)]`.

    Properties:
    - `private var state: RecordingSessionState = .idle`
    - `private var recorder: AudioRecorder?`
    - `private var startTime: Date?`
    - `private var pausedAt: Date?`
    - `private var totalPausedTime: TimeInterval = 0`
    - `private var pauseIntervals: [(start: TimeInterval, duration: TimeInterval)] = []`
    - `private var tempURL: URL?`
    - `private var destinationURL: URL?`

    Methods:
    - `var currentState: RecordingSessionState { state }`
    - `var elapsedSeconds: TimeInterval`: Compute from startTime minus totalPausedTime. If paused, freeze at last value.
    - `var currentLevel: Float`: Delegate to recorder.currentLevel. Return -160 if not recording.
    - `func startRecording(destination: URL) throws`: Guard state == .idle. Create temp URL in NSTemporaryDirectory(). Set startTime = Date(). Create AudioRecorder, call start(to: tempURL). Set state = .recording. Store destinationURL.
    - `func pause() throws`: Guard state == .recording. Call recorder.pause(). Record pausedAt = Date(). Set state = .paused.
    - `func resume() throws`: Guard state == .paused. Compute pauseDuration = Date() - pausedAt. Add to totalPausedTime. Record pause interval (start relative to recording start, duration). Call recorder.resume(). Set state = .recording.
    - `func stop() throws -> RecordingResult`: Guard state is .recording or .paused. Call recorder.stop(). Compute final duration. Move temp file to destinationURL via FileManager.moveItem (atomically if possible). Set state = .stopped. Return RecordingResult with audioURL, durationSeconds, pauseIntervals.
    - `func reset()`: Set state back to .idle, clear all stored data. For recovery after .stopped.

    Use `#if os(macOS)` guard. The actor is Sendable by Swift 6 actor isolation.

    Per P-D1: pauseIntervals are exposed so the UI/transcription layer can insert `[Paused HH:MM:SS-HH:MM:SS]` markers into the transcript.
    Per P-D2: temp-then-move via FileManager handles crash safety.
