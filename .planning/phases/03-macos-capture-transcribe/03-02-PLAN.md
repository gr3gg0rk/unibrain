  <behavior>

    - On download error or SHA256 mismatch: auto-retries once (P-18)

    - Model is stored at ~/Library/Application Support/Unibrain/models/ggml-small.en.bin (P-19)
    - Expected SHA256 hash is a static let constant embedded in the binary

    **SmallEnDownloader.swift:** Create `public actor SmallEnDownloader: Sendable` in Sources/UnibrainProviders/Transcription/ModelDownload/.
    - Define `public enum DownloadState: Sendable, Equatable`: `.notStarted`, `.downloading(Double)` (progress 0.0-1.0), `.verified`, `.failed(String)`.
    - `static let expectedSHA256: String` = the SHA256 hash of ggml-small.en.bin v1.7.4 (embed as string literal).
    - `static let downloadURL: URL` = `https://github.com/ggml-org/whisper.cpp/releases/download/v1.7.4/ggml-small.en.bin` (P-D6).
    - `static let modelStoragePath: URL` = `FileManager.default.url(for: .applicationSupportDirectory, in: .userDomainMask, appropriateFor: nil, create: true).appendingPathComponent("Unibrain/models/ggml-small.en.bin")` (P-19).
    - Properties: `private var state: DownloadState = .notStarted`, `private var retryCount = 0`, `private let maxRetries = 1` (P-18: retry once).
    - `var currentState: DownloadState { state }`
    - `var isModelPresent: Bool` — checks if model file exists at storage path AND passes SHA256 verification (used by UI to show readiness per P-10).
    - `func startDownload() async`:
      1. Guard state is .notStarted or .failed (allow retry from failed).
      2. Set state = .downloading(0.0).
      3. Try download via URLSession (use URLSession.shared.bytes(from:) for progress tracking, or URLSessionDownloadTask with delegate for background download).
      4. Write downloaded data to modelStoragePath. Create intermediate directories first (A-05 pattern from NoteWriter).
      5. Compute SHA256 via CryptoKit: `let hash = SHA256.hash(data: data)`, compare hex string to expectedSHA256.
      6. On success: state = .verified.
      7. On failure (download error OR checksum mismatch): if retryCount < maxRetries, increment retryCount and retry from step 2. If retries exhausted: state = .failed(errorDescription).
    - Use `import CryptoKit` for SHA256.
    - Use `#if os(macOS)` guard.
