    - Intermediate directories are created recursively before writing (A-05)

    - Write failures surface NoteWriterError with appropriate case (WRITE-06)
    - The note is serialized as: YAML frontmatter (via Yams) + "---\n" + title + body

  <action>
    Create `public struct NSFileCoordinatorNoteWriter: NoteWriter, Sendable` in Sources/UnibrainProviders/VaultWriting/. Use `#if os(macOS)` guard.

    `func write(_ note: NormalizedNote, to destination: URL) async throws`:
    1. Check for `.icloud` placeholder: if `destination.deletingPathExtension().appendingPathExtension("icloud")` exists (or `destination.path + ".icloud"` exists on iCloud), throw `NoteWriterError.iCloudPlaceholder` (WRITE-05).
    2. Create intermediate directories: `try FileManager.default.createDirectory(at: destination.deletingLastPathComponent(), withIntermediateDirectories: true)` (A-05). Catch and map to `.directoryCreationFailed`.
    3. Serialize the note: Use `Yams.YAMLEncoder` to encode `note.frontmatter` to YAML. Construct the full Markdown string: `"---\n" + yamlString + "---\n\n" + note.title + "\n\n" + note.body`. Encode as UTF-8 data.
    4. Write atomically using NSFileCoordinator:
       - Create `NSFileCoordinator(filePresenter: nil)`.
       - Call `coordinator.writeAccessAllowed = true`.
       - Use `coordinator.coordinate(writingItemAt: destination, options: .forReplacing, error: nil) { newURL in ... }`.
       - Inside the coordination block: write data with `data.write(to: newURL, options: .atomic)` (WRITE-04).
    5. Map any filesystem errors to NoteWriterError cases: `.permissionDenied`, `.diskFull`, `.alreadyExists`, `.underlying(error)`.
    6. The write is async — wrap NSFileCoordinator (which is synchronous/callback-based) in a continuation or run on a background queue via `Task.detached`.

    Import both Foundation and Yams at the top.
  </action>
