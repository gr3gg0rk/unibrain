  <action>
    Create `@Observable @MainActor final class MenuBarViewModel` in UnibrainApp/ViewModels/MenuBarViewModel.swift.

    **Published state (UI reads these):**
    - `var sessionState: SessionDisplayState` — enum mapping RecordingSession states + pipeline states to display states: `.idle`, `.recording`, `.paused`, `.transcribing`, `.completed`, `.error(String)`
    - `var elapsedTime: TimeInterval = 0` — updated via Timer.publish at 1Hz from RecordingSession.elapsedSeconds
    - `var micLevel: Float = -160` — updated via Timer.publish at 10Hz from RecordingSession.currentLevel
    - `var waveformBuffer: [Float] = []` — rolling buffer of last 64 mic level readings for waveform display (P-D5)
    - `var downloadProgress: Double? = nil` — nil when no active download or verified; Double 0.0-1.0 while downloading (P-17)
    - `var isModelReady: Bool = false` — true when model is verified or not needed (SpeechAnalyzer works without model)
    - `var pauseCount: Int = 0` — number of pauses in current session
    - `var totalPausedSeconds: TimeInterval = 0` — for paused-state summary line per UI-SPEC

    **Dependencies (injected):**
    - `private let session: RecordingSession`
    - `private let orchestrator: PipelineOrchestrator`
    - `private let downloader: SmallEnDownloader`
    - `private var pollTimer: Task<Void, Never>?`

    **Init:**
    - `init(session: RecordingSession, orchestrator: PipelineOrchestrator, downloader: SmallEnDownloader)` — stores deps, sets sessionState = .idle, starts polling download state.

    **Methods:**
    - `func startRecording() async` — Calls `try session.startRecording(destination: computeDestinationURL())`. Starts pollTimer (async Task that loops at 30fps reading session.elapsedSeconds and session.currentLevel, appending to waveformBuffer). Sets sessionState = .recording.
    - `func pauseRecording() async` — Calls `try session.pause()`. Stops pollTimer for mic level (keep timer display frozen). Increments pauseCount. Sets sessionState = .paused.
    - `func resumeRecording() async` — Calls `try session.resume()`. Restarts pollTimer. Sets sessionState = .recording.
    - `func stopRecording() async` — Calls `let result = try session.stop()`. Sets sessionState = .transcribing. Constructs PipelineInputs from result via PipelineWiring.makePipelineInputs. Launches `Task.detached(priority: .userInitiated)` that calls `try await orchestrator.run(inputs:)`. On completion: sets sessionState = .completed, fires UNNotification ("Lecture transcript ready" / "Opened in vault" per UI-SPEC). On error: sets sessionState = .error(message).
    - `func dismissCompletion()` — Resets sessionState = .idle, calls orchestrator.reset().
    - `func retryDownload()` — Calls `Task { await downloader.startDownload() }`.
    - `private func startPolling()` — Creates a Task that loops: read session.elapsedSeconds, session.currentLevel; update waveformBuffer (shift + append, max 64 elements). Run at ~30fps. Cancel when state leaves .recording.
    - `private func computeDestinationURL() -> URL` — Returns `~/Documents/Unibrain/lectures/YYYY-MM-DD-Lecture.m4a` using HardcodedVaultResolver path pattern (audio destination; the note path is resolved by the orchestrator).
    - `private func observeDownload()` — Task that periodically reads `downloader.currentState` and maps to downloadProgress/isModelReady.

    **Permission handling:**
    - `func requestMicrophonePermission() async -> Bool` — Checks/requests microphone permission. Per UI-SPEC, if denied: show "Microphone permission needed" in status line with link to System Settings.
