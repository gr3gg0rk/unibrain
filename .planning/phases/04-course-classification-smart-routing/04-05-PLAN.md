    **New methods**:

    `func requestCalendarPermission() async`:
    - Call `calendarProvider?.requestFullAccess()` or set permission to `.denied` on failure.
    - Update `calendarPermission` based on result.
    - Per P-02: fires on first recording alongside mic permission.

    `func checkCalendarPermission() async`:
    - Call `calendarProvider?.checkAuthorization()` and map to PermissionState.
    - Update `calendarPermission` property.

    `func handleClassificationPause(match: CourseMatch) async`:
    - Called when orchestrator reaches `.awaitingUserChoice`.
    - Creates a CoursePickerViewModel with mode based on match (.multiple or .none), loaded courses from CourseMappingStore, and recent codes.
    - Sets `overlayState = .coursePicker(mode)`.
    - The picker view calls `viewModel.selectCourse(...)` when user taps.

    `func selectCourse(_ selection: CourseSelection) async`:
    - Convert CourseSelection to a CalendarEvent for orchestrator.resume(with:).
    - For `.course(code)`: construct CalendarEvent(title: code).
    - For `.event(event)`: use the event directly.
    - For `.newCourse(code, name)`: construct CalendarEvent(title: code), and call courseMappingStore.upsert.
    - For `.skip`: call orchestrator.skipClassification().
    - Update CourseMappingStore: upsert mapping (M-03), addRecent (M-03).
    - Call orchestrator.resume(with: resolvedEvent).
    - Set overlayState = .none.

    `func skipClassification() async`:
    - Call orchestrator.skipClassification().
    - Set overlayState = .none.

    `func openSystemSettings()`:
    - macOS-only (`#if os(macOS)`): `NSWorkspace.shared.open(URL(string: "x-apple.systempreferences:com.apple.preference.security?Privacy_Calendars")!)`.
    - Per RESEARCH.md Pattern 5 and CONTEXT.md Claude's Discretion.

    `func setTerm(label:startDate:endDate:) async`:
    - Call courseMappingStore.setCurrentTerm.
    - Update currentTermLabel and termHasExpired.

    **Modify stopRecording()**:
    Before constructing PipelineInputs, check if calendar permission is granted. If so, fetch events from EventKitCalendarAdapter using the term range. If not, pass empty events array (manual picker will fire for .none). Add `requestCalendarPermission()` call before the first recording if permission is `.notDetermined`.

    The key change: after transcription completes and the orchestrator transitions to `.awaitingUserChoice` (observable via currentState polling or a callback), the view model calls `handleClassificationPause`.
