## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| iCloud Drive → .unibrain/consent.json | Consent state syncs between devices. |
| AuditTrailStore → Vault notes | Audit trail reads all .md files in vault. |
| NSWorkspace → Obsidian | Audit tab opens notes in external app. |
| mitmproxy → UnibrainApp | Network inspection tool intercepts HTTPS traffic for verification. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-06-29 | Tampering | iCloud consent conflicts | medium | mitigate | Atomic writes (.atomic option) prevent corruption. Schema version field enables future merge logic. Unit test verifies round-trip. |
| T-06-30 | Information Disclosure | Audit log PII leakage | low | accept | Audit log contains note paths (user's vault) and provider names (not secrets). No API keys or transcript content. Acceptable for MVP. |
| T-06-31 | Information Disclosure | Telemetry phone-home | critical | mitigate | Zero-telemetry verified via grep + mitmproxy (Task 4). No analytics SDKs in Package.swift. No outbound telemetry domains observed. |
| T-06-32 | Denial of Service | Audit trail scan performance | low | accept | Audit trail scan runs on-demand (user opens Audit tab). Vault scan is I/O-bound but acceptable. No background scanning. |
| T-06-33 | Tampering | FrontmatterSchema v1→v2 migration | low | mitigate | Additive change (optional fields). Decoder treats missing as nil. Unit test covers v1 decoding. No data loss. |
| T-06-34 | Tampering | Provider routing in offline mode | low | accept | Local-first path enforced by logic (provider check before network call). No silent fallback to cloud per DISC-05. |
</threat_model>
