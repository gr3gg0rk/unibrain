    **AuditTrailStore.swift** `parseAuditEntry(from:)` replaces the buggy single-line ternary `let status: AuditStatus = hasSummary ? .success : .success` with a 3-way `if / else if / else` block:
    1. If `hasSummary` (schema.summaryModel non-empty) → `.success` (the summary landed).
    2. Else if `usedLLMProvider` (schema.llmProvider non-empty) → `.failed` (cloud LLM was invoked but no summary survived — operation failed).
    3. Else → `.success` (local-only note; no cloud processing attempted; not a failure).

    `usedLLMProvider` is a new local `let` computed as `schema.llmProvider != nil && !schema.llmProvider!.isEmpty`. The function comment above the block documents all three branches so future readers do not re-introduce the copy-paste bug.
